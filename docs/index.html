Chrome Extension

Jira QuickView

Hover Jira issue keys on supported pages, open a rich issue popup, and perform Jira updates without leaving the page you are already reading.

Source Code Support Privacy Policy

What It Does

Jira QuickView detects Jira keys on enabled pages and shows issue metadata, descriptions, comments, attachments, history, and linked pull request context in a hover popup.

How It Connects

The extension uses your existing Jira browser session and sends requests directly from the browser to the Jira instance you configure.
